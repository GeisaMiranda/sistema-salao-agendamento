Título: Sistema de Gestão - Espaço Josy Miranda Estética
Link da Aplicação: https://sistema-salao-agendamento-1.onrender.com
Descrição: Este projeto consiste em uma aplicação web desenvolvida em Python utilizando o framework Flask, projetada para otimizar a gestão administrativa do "Espaço Josy Miranda Estética".
O sistema tem como objetivo principal centralizar o controle de operações do salão, permitindo que a proprietária realize o gerenciamento de forma rápida e segura através de uma interface web.
Principais Funcionalidades:
Login de Acesso: Área restrita para garantir a segurança dos dados do estabelecimento.
Gestão de Clientes: Cadastro, listagem e exclusão de clientes, mantendo os contatos organizados.
Gestão de Serviços: Controle dos procedimentos oferecidos pelo salão, com registro de nomes e valores.
Agendamento de Serviços: Módulo para criar novos agendamentos, vinculando clientes aos serviços desejados, com definição de data e hora.
